# Tech Stack

## Core Technologies
- **Language:** Rust (Edition 2021/2024)
- **GUI Framework:** Iced (v0.12)
- **Input Simulation:** rdev (v0.5.3) - *Note: Currently listed in Cargo.toml but hotkeys use global-hotkey.*
- **Global Hotkeys:** global-hotkey (v0.5)
- **Serialization:** Serde (v1.0), Serde JSON (v1.0)
- **System Paths:** dirs (v5.0)

## Key Libraries & Crates
- `iced`: Cross-platform GUI library inspired by Elm.
- `global-hotkey`: Cross-platform global hotkey handling.
- `rdev`: Library for listening and sending keyboard and mouse events (Clicking engine).

## Build Tools
- **Package Manager:** Cargo
- **Compiler:** rustc
